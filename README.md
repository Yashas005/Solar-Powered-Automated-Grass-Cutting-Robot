# Solar-Powered-Automated-Grass-Cutting-Robot
An IoT-based solar-powered autonomous grass cutting robot using Raspberry Pi 5, YOLOv5, OpenCV, and computer vision for obstacle detection.

Team Project

## Technologies Used

- Raspberry Pi 5
- Python
- OpenCV
- YOLOv5
- Flask
- GPIOZero
- Picamera2

## Features

- Solar-powered operation
- Autonomous grass cutting
- YOLO-based obstacle detection
- Live camera streaming
- Web interface for robot control
- Emergency stop feature
